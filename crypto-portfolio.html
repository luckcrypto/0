How it works

Self-custody, the way it should be

Most trackers make you hand over an account and your holdings to their servers. This one flips that: you keep everything, and cryptography keeps it honest.

🚫

No account, no server

There is no sign-up and no database. Your portfolio is built and calculated entirely in your browser. We never see a single number you enter.

💾

A file you own

Hit Save and you get a plain .txt backup on your device — yours to copy, move and store anywhere. Load it back any time to restore your portfolio exactly.

🔐

Encrypt & verify

Add a password and the file is encrypted with AES-GCM (key stretched via PBKDF2). Every file also carries a SHA-256 checksum, so a tampered or corrupted file is caught on load.

📡

Real prices only

Live data comes on demand from the public CoinGecko API. If it cannot load, you see a clear message — never invented numbers.

🧮

Honest P&L

Profit and loss use average-cost basis from your real transactions, with realized and unrealized split out so the picture is accurate, not flattering.

📦

Optional offline cache

Tick one box to keep a copy in this browser so your data survives a refresh. Untick it and nothing is stored locally — your call.

Important

Please read before you rely on this

A record-keeping tool — not financial advice, not a wallet

This is not financial, investment or tax advice. It tracks numbers you enter against public prices. Crypto is highly volatile and values can change fast; treat every figure as an estimate and do your own research before making decisions.

This is not a wallet and holds no funds or keys. Your backup file contains only your records — coins, amounts, dates and prices. It cannot move money and contains no private keys. Keeping it safe protects your privacy and records, not access to any crypto.

Self-custody means self-responsibility. There is no password reset and no recovery. If you encrypt a backup and forget the password, the data is unrecoverable — by anyone, including us. Keep more than one copy of important backups, and store any password securely.

Verify your backups. The SHA-256 checksum tells you a file is intact, but only you can confirm you saved the right one. Re-load a fresh backup occasionally to be sure it restores as expected.

Questions, answered

Portfolio tracker FAQ

Do I need an account to use this?
No. There is no account, login or sign-up. Everything runs in your browser; your portfolio exists only on your device and in the backup file you choose to save.
Is the SHA-256 part actually encryption?
No — SHA-256 is a one-way hash used as a tamper-evident checksum, so the tool can prove a loaded file was not changed. Real encryption is separate and optional: set a password and the file is encrypted with AES-GCM, with the key derived from your password via PBKDF2. No password means plain text plus a verifiable checksum.
What if I lose my file or password?
The data is gone — this is true self-custody with no server and no recovery. Keep more than one copy of your backup, and if you encrypt it, store the password safely. An encrypted file with a forgotten password cannot be recovered by anyone.
Does my data ever leave my device?
No. The only network request is to CoinGecko for public prices. Your holdings, transactions and backup file are processed entirely in your browser and never sent to us or any server.
Why is a coin I own missing a price?
Live prices load the top coins by market cap. A very small or new coin outside that set will still be tracked with your cost data, but may show no live price until it is covered. Your amounts and history are never lost.
